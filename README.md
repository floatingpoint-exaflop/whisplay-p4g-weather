# Whisplay Persona 4 Weather UI

A fan-made Persona-inspired weather and ambient UI built for the Whisplay display on Raspberry Pi systems.

This project combines:
- live OpenWeather integration
- animated startup sequences
- ambient weather audio
- Persona-style UI elements and layouts
- custom display/audio handling for the Whisplay hardware

## Disclaimer

Persona 4 and related visual/audio assets are property of ATLUS / SEGA.

This repository is a non-commercial fan project created for educational and personal enjoyment purposes. No ownership of original Persona assets is claimed.

## Hardware
- Raspberry Pi 500+
- Whisplay AI IO Display/Audio HAT
- WM8960 audio chipset

## Features
- Dynamic weather backgrounds
- Time-of-day transitions
- Ambient rain/thunder audio
- Persona-inspired UI overlays
- OpenWeather API integration
